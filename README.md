# CMP627 - Aprendizado Profundo

Este repositório contém implementações e experimentos desenvolvidos na disciplina de Aprendizado Profundo, do Programa de Pós-Graduação em Computação da Universidade Federal do Rio Grande do Sul (UFRGS).

O conteúdo aborda aplicações práticas e estudos relacionados a arquiteturas de redes neurais profundas, incluindo CNNs, DNNs, LSTMs, Autoencoders e Transformers.

## Tecnologias e Ferramentas

- Python
- TensorFlow
- Keras
- PyTorch
